GreenTech

E-Waste Management Application

Revolutionizing E-Waste Management

A streamlined solution connecting users with organizations to responsibly dispose of electronic waste. Built with React Native, TypeScript, and modern mapping technologies.

Launch Demo

Key Features

User Item Listing

Users can easily list electronic items for pickup with descriptions, images, and location information.

Organization Management

Organizations can view listed items, schedule pickups, and assign collectors to routes.

Interactive Maps

Integration with mapping services to optimize pickup routes and provide visual feedback.

Real-time Routing

Collectors can see optimized routes between pickup locations using OSRM integration.

User Profiles

Comprehensive user profiles with item history, point tracking, and account management.

Mobile-First Design

Crafted for optimal performance on mobile devices with responsive UI components.
